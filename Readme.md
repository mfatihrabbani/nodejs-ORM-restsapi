
# API SPECS
Membuat rest API menggunakan Sequelize sebagai ORM nya


## Create Product

Request :
- Method : POST
- Endpoint : "/api/products"
- Headers : 
    - Content-Type : application/json
    - Accept : application/json

- Body :

```json
{
    "id": "number"
    "name": "string",
    "price": "number",
    "description": "text"
}
```

Response: 
```json
{
    "code": "number",
    "status": "string",
    "data": {
        "id": "number",
        "name": "string",
        "price": "number",
        "description": "text"
    }
}
```
